# Project 2 — Personal Portfolio Website

A responsive personal portfolio website built as part of the **Frontend Development Internship at DecodeLabs (Batch 2026)**. Showcases an animated hero, about, skills, projects, education, and contact sections with smooth scroll-based interactions.

## ✨ Highlights

- Animated hero section with floating decorative elements
- Sticky header that updates style on scroll
- Scroll-spy navigation highlighting the active section
- Scroll-reveal animations for sections, skill bars, and animated counters
- Mobile sidebar navigation with hamburger menu and overlay
- Client-side contact form validation with inline error messages
- Keyboard-accessible interactive cards (`role="button"`)
- "Back to top" button

## 📂 Sections

- **Hero** — Intro and call-to-action
- **About** — Personal introduction
- **Skills** — Animated progress bars
- **Projects** — Featured work
- **Education** — Academic background
- **Contact** — Validated contact form

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (Vanilla) — Intersection Observer, DOM manipulation, form validation
